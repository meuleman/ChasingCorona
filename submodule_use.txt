
## First add a hook to the upstream (original) repository:
cd COVID-19
git remote add upstream https://github.com/meuleman/COVID-19.git
cd ..
## As per: https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/configuring-a-remote-for-a-fork

## Once this has been done, to update, do:
cd COVID-19
git fetch upstream
git checkout master
git merge upstream/master
cd ..
## As per: https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/syncing-a-fork
